# Тестовая реализация бота для контроля задач

## Первая реализация TelegramApi.py
Реализовано с помощью конечного автома, библиотеки requests для запросов к API телеграма.
Нет асинхронности. Сделано, чтобы понять принцип работы. Реализация с помощью ООП.

## Вторая реализация через библиотеку aiogram
Реализовано с помощью асинхронной билиотеки aiogram. Используются только асинхронные функиции, и один класс для состояния.
Так же реализована небольшая команда для получения рандомного котика благодря сервису *https://api.thecatapi.com/v1/images/search*
Реализована БД(sqlite) с таблицами для хранения пользователей и задач пользователей. Соответствующие методы представлены в файле database.py.

## Установка зависимостией
Созадать виртуальное окружение *python -m venv venv*  
*pip install --upgrade pip*  
*pip install -r req.txt*  


## Запуск бота
*Вариант 1* через docker-compose.yml
  - docker compose up - t build
*Вариант 2* через запуск *python3 src.main.py*
